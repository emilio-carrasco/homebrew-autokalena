# Homebrew Tap — AutoKalena

Instala la app de escritorio **AutoKalena** en macOS con Homebrew:

```bash
brew install --cask emilio-carrasco/autokalena/autokalena
```

La app no está firmada (sin cuenta de Apple Developer); el cask retira la cuarentena de
Gatekeeper tras instalar. Fuente pública de descargas:
<https://github.com/emilio-carrasco/autokalena>.
